# FadDompet v1.3.0

Rilis ini berfokus pada peningkatan Analytics, konsistensi icon kategori, sinkronisasi budget, dan kesiapan rilis versi 1.3.0.

## Pembaruan

- Menambahkan filter periode Analytics.
- Memperjelas fungsi grafik dengan keterangan yang lebih informatif.
- Menampilkan nominal dan persentase pada legend pengeluaran per kategori.
- Memperbaiki sinkronisasi informasi budget antara Dashboard dan Analytics.
- Memperbaiki tampilan form budget agar lebih konsisten dengan UI FadDompet.
- Memperbaiki icon kategori agar lebih sesuai konteks.
- Memperbaiki tampilan versi aplikasi agar mengikuti versi build secara otomatis.
- Merapikan dokumentasi dan template GitHub untuk v1.3.0.

## Download

Untuk sebagian besar HP Android modern, gunakan:

`FadDompet-v1.3.0-arm64.apk`

## Catatan

FadDompet berjalan offline, tanpa akun, tanpa cloud sync, tanpa iklan, dan tanpa tracking. Data disimpan lokal di perangkat.

File cadangan berisi data keuangan, jadi simpan di tempat yang aman.

Jika memasang APK dari luar Play Store, Android atau Play Protect mungkin tetap menampilkan peringatan.
